طريقة الدمج

أضف MailFlow لموقعك في دقيقتين — بديل كامل لـ EmailJS

1. أضف SDK عبر HTML

<script src="https://your-api.com/sdk.js"></script>

أو عبر npm

npm install mailflow-js

2. الكود الكامل (مثال نموذج تواصل)

// تهيئة المكتبة MailFlow.init('pk_your_key_here') // إرسال عند إرسال النموذج document.querySelector('#contact-form').addEventListener('submit', async (e) => { e.preventDefault() const result = await MailFlow.send({ template_id: 'your_template_id', to_email: 'you@example.com', params: { name: e.target.name.value, email: e.target.email.value, message: e.target.message.value } }) if (result.ok) alert('✅ تم الإرسال!') else alert('❌ فشل: ' + result.error) })

3. REST API مباشرة (curl / fetch)

curl -X POST https://your-api.com/v1/send \ -H "Content-Type: application/json" \ -H "X-Public-Key: pk_your_key" \ -d '{"template_id":"tpl_id","to_email":"test@example.com","params":{"name":"أحمد"}}'
